A distribution is an entity in pulp representing kickstart tree metadata/files associated to a repo.

Base URL Path: /distribution/

List Distributions by Repository

List available distribution for a given repo. This will return a list of all distributions associated to that specific repoid. The list is empty if no distributions are found.

Status: complete

Path: /repositories/<id>/list_distribution

Method: GET

Parameters:

repoid : repo id for which distribution is queried

Authentication:

username/pass or Consumer Certificate

Filters: None

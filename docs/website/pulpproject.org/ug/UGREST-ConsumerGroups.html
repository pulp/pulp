Method: PUT

username/pass

None

Headers:

  'id': 'mygroup',

Install Package

Request that package(s) be installed on a consumer group.

method: POST

path: /consumergroups/<group id>/installpackages/

permission: EXECUTE

success response: 202 ACCEPTED

failure response:

  • 404 Not Found if there is no such group

return: job ID

parameters:

  • packagenames <list> a list of package names
  • scheduled_time <str> (optional) schedule in iso8601 format; immediate when Null
